## 🧪 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Languages:** HTML, CSS, TypeScript
- **Styling:** Tailwind CSS (or module CSS if applicable)
- **Backend:** In progress (to handle user auth, profile updates, and image uploads)
- **Icon:** Custom favicon located at `public/favicon.io`

---

## 🚀 Getting Started

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/Lincoln-Madaraka/bloodafya.git

# Navigate into the directory
cd bloodafya

# Install dependencies
npm install

# Start the development server
npm run dev

The app will be available at http://localhost:3000.
